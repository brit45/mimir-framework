
# Mímir Framework

Version: 2.3.0

But: Mímir est un framework de deep learning haute performance en C++17, conçu pour créer, entraîner et exécuter des modèles (vision/NLP/diffusion) principalement sur CPU avec optimisations matérielles (SIMD/OpenMP), une API de scripting Lua et un système de sérialisation de checkpoints (dont SafeTensors), avec accélération Vulkan Compute optionnelle.
